# PL/I and Db2 Quiz System (AI-Powered)

An interactive **Tkinter-based quiz application** that generates **PL/I and Db2 technical questions using OpenAI**.  
Designed for mainframe developers, trainees, and educators, the system supports multiple difficulty levels and produces detailed, exportable result reports.

## Features
- GUI with Tkinter
- AI-generated PL/I and Db2 questions
- Difficulty levels: Junior to Senior
- Automatic scoring and grading
- Export results to TXT and JSON

## Requirements
- Python 3.7+
- tkinter
- openai (`pip install openai`)
- OpenAI API key (`OPENAI_API_KEY`)

## Running
```bash
python quiz_system.py
```
